# Camping MCP 🏕️
A MCP server designed to help you plan your next camping trip.

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

## Running This Project
To run this you **MUST** have uv installed. Follow the instructions [here](https://docs.astral.sh/uv/getting-started/installation/) to get uv installed on your system.
### Codex
```bash
codex mcp add <NAME> -- uv run --project <PATH TO YOUR PROJECT> fastmcp run <PATH_TO_MAIN.PY>
```
